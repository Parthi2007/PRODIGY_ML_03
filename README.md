# Cats vs Dogs Classification using Support Vector Machine (SVM)

This project is part of my Machine Learning Internship at Prodigy InfoTech.

## Project Objective

The objective of this project is to implement a Support Vector Machine (SVM) model to classify images of cats and dogs using image processing and machine learning techniques.

## Dataset

Dogs vs Cats Dataset from Kaggle:

https://www.kaggle.com/datasets/chetankv/dogs-cats-images

## Technologies Used

* Python
* NumPy
* OpenCV (cv2)
* Matplotlib
* Scikit-learn
* Google Colab

## Machine Learning Concepts Used

* Supervised Learning
* Image Classification
* Support Vector Machine (SVM)
* Image Preprocessing
* Feature Extraction
* Model Evaluation

## Project Workflow

1. Load dataset images
2. Resize images to 64x64
3. Convert images into numerical arrays
4. Label images as cats or dogs
5. Split dataset into training and testing sets
6. Train SVM classification model
7. Predict image classes
8. Evaluate model accuracy
9. Visualize predictions

## Model Details

SVM Kernel Used:

```python
kernel='linear'
```

## Output

The model successfully classifies images of cats and dogs and displays prediction results with image visualization.

## Accuracy

Achieved approximately 64% accuracy using SVM on resized image data.
